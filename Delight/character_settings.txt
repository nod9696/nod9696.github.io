# Project Delight キャラクター設定書
# 「双極のグリモワール」

---

## 世界観

舞台：ファーリア王国 自治区
背景：大国に囲まれた貧しい小国。傭兵稼業を主産業とし、間接的武力で自国を守る。
地下迷宮：ファーリア国内で発見された巨大地下遺構。エーテル（魔力エネルギー）が満ちている。
　　　　　地上では魔法が使えないが、迷宮内では使用可能。
　　　　　深層に留まると「変質」が起き、人間から逸脱する。
グリモワール：迷宮最深部に眠る二冊の魔導書（白・黒）。「全ての願いが叶う」とされる。
星の意思：太古に地球に巣食った巨大エーテル塊。迷宮の元凶。

## 主要キャラクター

---

### ライ（主人公）

**基本情報**
- 性別：男 / 年齢：17〜18歳 / 武器：片手剣（軽量型）
- 一人称：俺 / 口調：淡々、内省的。短文。
- 口癖：「……そうか」（受容の言葉）
- 外見：特筆なし（読者の投影用）
- 遺品：母の手縫いのお守り袋（紺色、不揃いな星の刺繍）。中身不明。「開けるのはまだ早い」

**このキャラを好きになる理由**
ライの魅力は「空っぽだからこそ、全てが染みる」ところにある。彼は何も持っていない。才能も大志も、帰る場所もない。しかしその空洞が、出会う人々の温度をまっすぐ受け止める器になっている。メリルの声がやけに響くのも、クレアの祈りが胸に刺さるのも、彼が空っぽだからだ。読者はライの空洞に自分を重ね、彼と一緒に少しずつ満たされていく体験をする。

**欠落と闇**
- 根本的欠落：「生きる理由がない」。母の死後、世界との接続が切れている。
- 闇の正体：自分の命を軽く見ている。野垂れ死に覚悟は、裏を返せば自己放棄。
- 無意識の罪悪感：母の最期に何もできなかった自分を許せていない。

**固有能力：「気づく力」**
- 壁の苔の乱れ、空気の変化など微細な異変を無意識に察知する
- 戦闘中の「潮目」を感じ取る（体がついてこないのが課題）
- ヴァイ評：「体力はないが目はいい。動く前に見えてる」
- この力の本質：他者の痛みに気づく力でもある。猫をやるヴァイ、外套をかけるルーミア、飾り紐に触れるミト――小さな仕草の裏にある感情を見抜く。

**キャラクターアーク**
| フェーズ | 内面 | 象徴 |
|---|---|---|
| 序盤 | 「何のために起き上がるのかわからない」 | お守り袋に無意識に触れる |
| 中盤 | 「この連中と一緒にいたい」（自覚なし） | 仲間の異変に気づき始める |
| 繭戦 | 「俺がやらなければ」（初めての能動性） | 剣を握る手の震えが力に変わる |
| 終盤 | 「戻る場所ができた」 | クレアの「無事に戻って」に頷ける |
| ルート分岐後 | 仲間のために戦う覚悟 | お守り袋を開ける（各ルートのクライマックス） |

### メリル

**基本情報**
- 性別：女 / 年齢：15〜16歳 / 武器：魔導杖（赤いリボン付き）
- 一人称：あたし（日常）→ わたし（変質後）
- 口調：「〜じゃん！」「〜でしょ！」上げ調子。早口。
- 口癖：食べ物で何でも喩える（唐揚げ、天ぷら、メロンパン、グラタン、ホットケーキ等）
- 外見：桃色の髪。丸い目。小柄。

**このキャラを好きになる理由**
メリルは「読者を笑わせながら泣かせる」キャラだ。唐揚げやメロンパンの喩えで笑わせておいて、それが恐怖を飼い慣らすための防衛機制だと気づいた瞬間、笑いが愛おしさに変わる。15歳の少女が地下迷宮でモンスターと戦いながら「グラタンみたい」と言っている――その明るさの裏にある必死さに、読者は心を掴まれる。

**欠落と闇**
- 根本的欠落：「自分一人では怖くて立てない」。魔法科主席のプライドと、兄がいないと崩れる脆さ。
- 闇の正体：食べ物で喩えることは、現実を直視できないことの裏返し。真正面から恐怖を見つめたら、この子は折れる。
- 寝ている間にエーテルが漏れ出す体質＝力をコントロールしきれていない。才能が自分の器を超えている不安。

**固有クセの意味**
- 食べ物喩え：恐怖→食欲に変換する生存本能。楽しそうに見えて、実は必死。
- 「あたし」→「わたし」：MagicaToxinルートで変質が進むにつれ一人称が変化。本人も気づかない。読者だけが変化の兆候を追える。

**キャラクターアーク**
| フェーズ | 内面 | 象徴 |
|---|---|---|
| 序盤 | 天真爛漫の鎧を着て戦場に立つ少女 | 「唐揚げみたい！」 |
| 中盤 | 兄への依存と自立の間で揺れる | ルーミアがいない場面で怯える |
| 暗礁帯 | 鎧が剥がれる。初めて「怖い」と認める | 泣きながら兄にしがみつく |
| MagicaToxin | 変質しながらも「あたし」を保とうとする | 一人称が「わたし」に変わる |
| クライマックス | 変質を受け入れるか拒否するかの選択 | 「わたしがわたしであることは、変わらない」 |

### ルーミア

**基本情報**
- 性別：男 / 年齢：18〜19歳 / 武器：長弓、鉄盾
- 一人称：俺 / 口調：短文、体言止め。必要最低限。
- 口癖：「そうか」「食べすぎ」「たまたまだ」
- 外見：長身、細身だが芯がある体格。

**このキャラを好きになる理由**
ルーミアの魅力は「言わない愛」にある。蜜漬けの林檎を「たまたまだ」と言って差し出す。外套を黙ってかける。ホットケーキを焦がす。言葉にしない分だけ、行動の一つ一つが雄弁で、読者は彼の不器用な優しさを「見つける」快感を得る。作中で最もセリフが少ないのに、最も多くを語っているキャラ。

**欠落と闘**
- 根本的欠落：「感情の出し方を知らない」。感じていないのではなく、表現回路が未発達。
- 闇の正体：メリルを守ることが人生の全てになっている。自分自身の人生がない。メリルがいなくなったら、この男は空洞になる。
- 料理が下手＝不器用さの象徴。愛情はあるのに、形にする技術がない。

**キャラクターアーク**
| フェーズ | 内面 | 象徴 |
|---|---|---|
| 序盤 | 妹を守る装置として機能している | 無言で外套をかける |
| 中盤 | 仲間との交流で「妹以外」の世界に触れる | ヴァイと無言で酒を飲む |
| 暗礁帯 | メリルを守れなかった恐怖 | 盾のように妹の前に立つ |
| MagicaToxin | メリルの変質を止められない無力感 | 初めて叫ぶ（感情爆発） |
| クライマックス | 守るだけでなく「信じる」ことを選ぶ | メリルの選択を尊重する |

### ヴァイ（通称クロキ）

**基本情報**
- 性別：男 / 年齢：20代前半 / 武器：二本の短剣
- 一人称：俺 / 口調：ぶっきらぼう、最短。
- 口癖：「やめとけよ」「ああ」「面白い」
- 外見：大柄、がっしり。古い傷跡多数。左腕に古い火傷。

**このキャラを好きになる理由**
ヴァイは「強さの中にある優しさ」で読者を掴む。一言で巨漢を黙らせるカリスマがありながら、足元の痩せた野良猫に干し肉をやる。見つかると耳が赤くなる。この「ギャップ萌え」がラノベ/アニメ的な快感を生む。強い男が弱い存在に見せる柔らかさは、最強のキャラ造形の一つ。

**欠落と闇**
- 根本的欠落：「強さ以外で自分を定義できない」。戦い続けることでしか存在意義を感じられない。
- 闇の正体：左腕の火傷。過去に守れなかったもの。戦闘を楽しむのは、あの日の無力感から逃げ続けているから。
- フレデリクとの友情が彼の唯一の錨。それを失った時、ヴァイは何になるか。

**キャラクターアーク**
| フェーズ | 内面 | 象徴 |
|---|---|---|
| 序盤 | 純粋な戦闘狂に見える | 「面白い」と呟く |
| 中盤 | ライを指導する中で「教える喜び」を知る | 「お前は化ける」 |
| 日常 | 猫への優しさが漏れる | 干し肉をこっそり落とす |
| Dominater | フレデリクの裏切りで唯一の錨を失う | 一瞬だけ動きが止まる |
| クライマックス | 「強さ以外の理由」で剣を握る | 仲間を守るために戦う |

### フレデリク

**基本情報**
- 性別：男 / 年齢：20代前半 / 武器：剣、医療知識
- 一人称：私 / 口調：丁寧語、穏やか。
- 口癖：（眼鏡を拭きながら）「〜ですね」
- 外見：丸い眼鏡、修道士風衣装、温厚な顔立ち。

**このキャラを好きになる理由**
フレデリクの魅力は「善意が本物であること」にある。孤児への支援も、仲間への気遣いも、全て本心。しかし彼は同時に死の商人でもある。この矛盾を読者が知った時、「なぜ？」という疑問が生まれ、その答え――善意だけでは何も守れなかった少年の絶望――を知った時、嫌いになれなくなる。最も許せない裏切り者であると同時に、最も同情されるキャラ。

**欠落と闇**
- 根本的欠落：「善意が無力であるという絶望」。貧民街で善意だけでは誰も救えなかった原体験。
- 闇の正体：善良な自分を演じているのではない。善良な自分が「足りない」ことを知っている。だから裏の手段を持った。善と悪の共存ではなく、善が悪を産んだ構造。
- 眼鏡を拭う仕草＝自分を整える儀式。曇りのない視界を保つこと＝自分の善性が曇っていないか確認し続ける行為。

**眼鏡バロメーター**
- 平静：ゆっくり丁寧に拭く
- 興奮/緊張：拭く間隔が短くなる
- 動揺：拭わない（自分を整える余裕がない）
- 裏切りの瞬間：眼鏡を外して踏み砕く（＝善性の仮面を自ら破壊する）

**キャラクターアーク**
| フェーズ | 内面 | 象徴 |
|---|---|---|
| 序盤 | 完璧な善人。誰からも信頼される | 眼鏡をゆっくり拭く |
| 中盤 | 仲間との絆が深まり、裏の仕事との矛盾が増す | 拭く間隔が短くなる |
| リベル編 | 武器の流れを追う中で伏線が散る | 商人の自殺に眼鏡を拭わない |
| Dominater | 正体が明かされる | 眼鏡を踏み砕く |
| クライマックス | 「お前だけは巻き込みたくなかった」 | ヴァイとの最後の対話 |

### クレア

**基本情報**
- 性別：女 / 年齢：17〜18歳 / 武器：回復魔法
- 一人称：私 / 口調：丁寧、やわらかい。
- 口癖：「無事に戻ってきてくださいね」
- 外見：黒髪を結い上げる。シスターの白い衣。荒れた手。

**このキャラを好きになる理由**
クレアの魅力は「揺るがない祈り」にある。自分の食事を後回しにし、自分の傷を後回しにし、子供の絵を宝物と呼ぶ。この自己犠牲は美しいが、同時に危うい。「自分を大切にしない人は、いつか壊れる」という予感が読者の心に刺さる。そして実際に壊れる（Patagonistルート）。だからこそ、壊れる前の彼女の祈りが、痛いほど愛おしい。

**欠落と闇**
- 根本的欠落：「自分の価値を他者への奉仕でしか測れない」。孤児として育ち、居場所を「作る」ことでしか安心できない。
- 闇の正体：信仰への依存。「神様が見ていてくださる」は救いであると同時に、自分で判断することからの逃避。神の沈黙が続いた時、彼女の拠り所が崩壊する。
- 三層で子供のモンスターが弾け飛ぶシーン＝彼女の信仰の土台が揺らぐ瞬間。「なんで治らないの！？」は神への初めての叫び。

**「無事に戻って」の深化**
1回目：患者への儀礼的な声かけ → 2回目：ライ個人への願い → 3回目：祈りそのもの
→ Patagonistルートでラスボス化後、同じ台詞を狂った笑みで言う
＝祈りが呪いに反転する瞬間

**キャラクターアーク**
| フェーズ | 内面 | 象徴 |
|---|---|---|
| 序盤 | 穏やかな信仰者。揺るがない祈り | 「その方が、明日も生きていますように」 |
| 中盤 | 孤児たちの世話を通じてライとの絆 | 子供の絵、荒れた手 |
| 三層 | 子供のモンスターが弾け飛ぶ。信仰が揺らぐ | 「なんで治らないの！？」 |
| Patagonist | 星の底でエーテルに呑まれる | 「神様が居ました」（狂気の笑み） |
| 対決 | 祈りが呪いに反転した姿 | 「無事に戻ってきてくださいね」（壊れた声で） |

### ノア

**基本情報**
- 性別：女 / 年齢：16〜17歳 / 武器：大槍（白い装飾）
- 一人称：私 / 口調：凛。命令形も丁寧。
- 口癖：「慣れています」「時間を稼いでください」
- 外見：端正。白と金のドレス。長身。

**このキャラを好きになる理由**
ノアの魅力は「完璧な鎧の隙間から見える脆さ」にある。戦闘では無双。三秒で三人を蹴散らす。しかし道を間違える。「友達」の意味がわからない。ホットケーキを食べたことがない。ドレスの裾を無意識に握る。この「最強×世間知らず」のギャップが読者の保護欲を刺激する。彼女を守りたいのではない。彼女に「普通」を教えたくなる。

**欠落と闇**
- 根本的欠落：「人との繋がり方を知らない」。レゾネイターとして幼少から戦い続け、同年代の友人がいたことがない。
- 闇の正体：「慣れています」は万能の防壁。戦いにも、孤独にも、痛みにも慣れている。しかし慣れることは克服ではない。痛みを感じなくなっているだけ。
- ドレスの秘密：母から貰った唯一の私服。母は既に亡い。このドレスを着て戦うのは、母の記憶と一緒に戦っているから。汚れても気にしないのは、母との絆は汚れないと信じているから。

**キャラクターアーク**
| フェーズ | 内面 | 象徴 |
|---|---|---|
| 登場 | 完璧な戦の女神。隙がない | 三秒で三人を蹴散らす |
| 帰路 | 方向音痴が露見。鎧に亀裂が入る | 「逆だよノアちゃん」 |
| メリルとの交流 | 「友達」の意味を初めて考える | ドレスの裾を握る手が震える |
| 日常 | ホットケーキを食べる。「おいしいです」 | 普通の感情を知り始める |
| Patagonist | クレアと共に星の底へ。力の意味を問われる | ドレスが母の形見だと告白 |

### エト

**基本情報**
- 性別：男 / 年齢：14〜15歳 / 武器：長弓
- 一人称：私/僕 / 口調：のんびり、敬語混じり。
- 口癖：「あねさま」「大したことじゃないですよ」
- 外見：青い髪。細身。ミトと瓜二つだが柔らかい。

**このキャラを好きになる理由**
エトの魅力は「天然の中に眠る凄み」にある。酒場を三周する方向音痴。ミトにご飯を食べさせてもらう甘えん坊。鼻歌を歌うぼんやり少年。しかし弓を構えた瞬間、水面に氷が張るように全てが変わる。このギャップは「本当の強さは普段見えない」というラノベ的快感の王道。加えて、彼の無自覚な信頼（「あねさまが言うなら大丈夫」）が、ミトの裏切りの悲劇性を極限まで高める。

**欠落と闇**
- 根本的欠落：「自分の運命を知らない」。神への供物として差し出される運命を、姉が隠している。
- 闇の正体：エーテル感化体質。制御できず高熱とトランス状態を引き起こす。体が自分のものではない感覚。
- 鼻歌の正体：古代の呪歌。本人は子守歌だと思っている。この歌がエーテルを鎮める力を持っていること＝エトは無自覚に自分の運命と戦っている。

**キャラクターアーク**
| フェーズ | 内面 | 象徴 |
|---|---|---|
| 序盤 | 天然ボケの甘えん坊。姉に全幅の信頼 | 「むぐむぐ、ありがと、あねさま」 |
| 中盤 | 発作が増える。鼻歌が無意識に出る頻度も増す | トランス状態で歌う |
| 薬の真実 | ミトの薬が自分を変えていると薄々気づく | 「あねさま、僕はどうなるんですか」 |
| Criminal Mariage | 人造グリモワールと化す | 異形の中で鼻歌を歌い続ける |
| クライマックス | 姉の愛と裏切りを同時に理解する | 「あねさまは、ずっと泣いていたんですね」 |

### ミト

**基本情報**
- 性別：女 / 年齢：16〜17歳 / 武器：長弓（飾り紐付き）
- 一人称：なし / 口調：最低限。無感情。
- 口癖：「あにさま、口開けて」「そこがいのちか」
- 外見：青い髪。長身。エトと瓜二つだが硬い。

**このキャラを好きになる理由**
ミトの魅力は「氷の下の灼熱」にある。表面は完全な無感情。最低限の言葉しか発しない。しかしその全ての行動が「エトを守る」という一点に集約されている。弓の飾り紐に触れる祈りの仕草。エトが歌う時だけ僅かに抜ける肩の力。読者は彼女の微細な感情の揺れを「発見」することで、氷の下に隠された灼熱の愛を知る。そしてその愛が「裏切り」の形をとっていると知った時、読者は泣く。

**欠落と闇**
- 根本的欠落：「掟に勝てない」。北国の掟ではエトは神への供物。正面から掟を壊す力がないから、逃走を選んだ。
- 闇の正体：エトを触媒にしようとしている自分を、彼女自身が許せていない。しかし他に道がない。「永劫の逃避行」は美しい言葉だが、裏を返せば「永遠に逃げ続ける」ことしかできない自分への呪い。
- 弓の飾り紐＝北国の神への祈り。彼女は弟を供物にしようとする神に祈っている。矛盾。しかし祈ることしかできない。

**キャラクターアーク**
| フェーズ | 内面 | 象徴 |
|---|---|---|
| 序盤 | 完璧な姉。無表情でエトの世話をする | 「口拭いて」 |
| 中盤 | 薬を投与し続ける。表情は変わらない | 飾り紐に触れる頻度が増す |

| クライマックス | エトの異形化。計画の破綻 | 初めて泣く。声を出さずに。 |

### ナホァ

**基本情報**
- 性別：女 / 年齢：不詳（20代前半？） / 武器：格闘術（関節技）
- 一人称：うち/私 / 口調：「〜ナ」語尾、砕けた丁寧語。
- 口癖：「お料理！」→（ゴキ）→「どうしましたお客さん？」
- 外見：長い黒髪、整った顔立ち、底の知れない目。

**このキャラを好きになる理由**
ナホァの魅力は「底が見えない」ことそのもの。読者は彼女を最後まで掴めない。天真爛漫な「お料理！」と腕を折る暴力の落差。営業スマイルと値踏みする目の温度差。黒い冗談のセンス。全てが「この女、何者だ？」という問いを生み続ける。ミステリアスな女性キャラの王道にして、物語の全体構造を握る「観察者」。彼女の真意が明かされた時、物語の全景が見える。

**欠落と闇**
- 根本的欠落：「使命に人生を奪われた」。師匠の遺志を継ぎ、星の意思を止める使命を背負っている。自分の人生を生きたことがない。
- 闇の正体：人を「道具」として見る訓練を受けてきた。ライハを変質させる計画も、最初は任務として割り切っていた。しかしライハと過ごす日々が彼女を変えた。「もう道具には見えなかった」。
- 笑顔の使い分け＝感情を制御する技術。本当の笑顔がどれか、本人にもわからなくなっている。

**キャラクターアーク**
| フェーズ | 内面 | 象徴 |
|---|---|---|
| 序盤 | 酒場の給仕。全員を観察している | 微笑みの奥の別温度の目 |
| 中盤 | ライハに接近。計画的に距離を詰める | 「おにいさん、骨あるんだ」 |
| Murder | ライハを変質させるか葛藤 | ライハの寝顔を見つめるシーン |
| 真実 | 師匠の遺志と自分の感情の衝突 | 「あの人に、帰ってきてほしい」 |
| クライマックス | 初めて本当の顔を見せる | 「泣いているような笑い」 |

### ライハ・三島

**基本情報**
- 性別：女 / 年齢：18〜19歳 / 武器：東方の長刀
- 一人称：うち / 口調：「〜だよね！」豪快。
- 口癖：「肉！ 肉肉！ あと酒！」
- 外見：東方の衣装、長刀を背負う。

**このキャラを好きになる理由**
ライハの魅力は「豪快さの裏にある傷」だ。三人前食べて酒を飲んで笑う。距離が近くて酔っ払い。ラノベ的な「うざ可愛い」の権化。しかし「寂しいからいっぱい食べるんだよ」の一言で全てがひっくり返る。そして夜、一人で刀の手入れをする横顔。炉の残り火に照らされた、年相応の少女の顔。この落差で読者は撃ち抜かれる。

**欠落と闇**
- 根本的欠落：「父の影から出られない」。剣の構え、踏み込み、間合い、全てが父の鏡写し。自分固有のものが何もない。
- 闇の正体：父を殺さなければならないかもしれない、という予感。しかし父の技で父を斬ることは、自分を斬ることでもある。
- 大食い＝東方の村での飢餓体験。食える時に食う。体が飢えを覚えている。豪快さではなく、サバイバル本能。

**キャラクターアーク**
| フェーズ | 内面 | 象徴 |
|---|---|---|
| 序盤 | 豪快な女剣士。全てを笑い飛ばす | 「肉！ 肉肉！」 |
| 酔い | 鎧が緩む。父の話を漏らす | 「殺人鬼って多分さぁ、うちの親父なんだよね！」 |
| 夜 | 一人で刀を磨く。祈りのような手つき | 炉の残り火に照らされた少女の横顔 |
| 父との対峙 | 鏡合わせの技。同じ構え、同じ踏み込み | 「助太刀ありがたい！ こいつの力、人間離れしてる」 |
| クライマックス | 父の刀を受け取る。鞘で打つ | 「じじぃの刀が泣いてる」 |

### ロイド

**基本情報**
- 性別：男 / 年齢：20代半ば / 武器：騎士剣
- 一人称：俺/私（公式） / 口調：簡潔、軍人口調。
- 口癖：「飛び込め」
- 外見：短髪、鋭い目。騎士団の鎧。

**このキャラを好きになる理由**
ロイドの魅力は「正しさの暴走」にある。彼の怒りは正当だ。傭兵稼業で使い捨てにされる兵士たち。父の死。国の欺瞞。彼が間違っているのは方法だけで、動機は正しい。読者は彼を倒すべき敵だと思いながら、心のどこかで共感してしまう。「この人の言うことは正しい。でも止めなければならない」。その葛藤が物語に深みを与える。

**欠落と闇**
- 根本的欠落：「怒りを手放せない」。父の死への怒りが人生の推進力になっている。怒りがなくなったら、この男は空っぽになる。
- 闇の正体：国を変えたいのか、復讐したいのか、自分でもわからなくなっている。

### ミレイ

**基本情報**
- 性別：女 / 年齢：20代前半 / 武器：剣
- 一人称：私 / 口調：事務的。「……」多用。
- 外見：詳細不明。静かで透き通った印象。

**このキャラを好きになる理由**
ミレイの魅力は「不在の感情」にある。物語を通して一度も笑わない。感情が見えない。事務的に話し、事務的に戦う。しかしロイドの前でだけ、声が僅かに柔らかくなる。読者はその「僅か」を見逃さない。そして最期の瞬間、初めて笑う。「やっと、終わりますね」。物語で唯一の笑顔が、死の瞬間に現れる。この構造が読者の心を抉る。

**欠落と闇**
- 根本的欠落：「ロイド以外の世界が存在しない」。彼のためだけに生き、彼のためだけに死ぬ。自分の意志がどこにあるのか、本人にもわからない。
- 闇の正体：ロイドの反乱が間違いだと知っていた。知っていて止めなかった。止める理由がなかった。ロイドが望むなら、それでいい。その盲従が、最大の愛であり最大の罪。

## 全キャラ共通の設計原則

1. **欠落が魅力を生む**：完璧なキャラはいない。全員が何かを失っている。その欠落が、他者を必要とする理由になり、関係性を生む。
2. **表と裏の落差**：日常で見せる顔と、極限で見せる顔の落差が大きいほどキャラは魅力的になる。
3. **小さな仕草に全てを込める**：台詞ではなく仕草で語る。眼鏡を拭く、飾り紐に触れる、干し肉を落とす。読者が「発見する」快感を設計する。
4. **ラスボス化する仲間**：全ルートで仲間がラスボスになる構造。日常の積み上げが大きいほど、その崩壊の衝撃が大きい。
5. **アークの到達点は「選択」**：各キャラクターの成長は、最終的に「選択」に帰結する。変質を受け入れるか。裏切りを許すか。父を斬るか。その選択の重さが、物語の価値を決める。
