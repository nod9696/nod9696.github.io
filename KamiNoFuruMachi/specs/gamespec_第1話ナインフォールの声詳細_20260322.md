# 📖 シナリオ仕様（第1話）

## 第1話 概要

**タイトル**：第1話「ナインフォールの声」
**舞台**：迷宮都市ナインフォール・廃区（「傷区」）
**物語の核心**：カナタがリリスと初めて出会う。リリスはウタに引かれ廃区に向かっていた。マヨイからリリスを守る。カナタはリリスを**「名前も知らない少女」として扱う**（名前を聞いても覚えない/呼ばない）。

> ⚠️ **引継ぎ資料より**：カナタがリリスの名前を初めて呼ぶのは第6話。第1話では例え名前を聞いても、カナタの内心では「少女」として認識する。

## ウタ（神の声）の音響仕様

```
コンセプト: "Vocality Without Source"（発声源のない声性）

FMOD DSP処理チェーン（DSP_UTA_CHAIN）:
  [1] Pitch Shifter: ±7semitone（param_mayoi_level連動）
  [2] Multi-tap Delay: 47ms/-6dB, 113ms/-12dB, 271ms/-18dB
  [3] Convolution Reverb: 神殿廃墟IR 40% + 頭蓋内腔IR 20%
  [4] Bitcrusher: param_mayoi_level > 0.6のみ / 16→10bit
  [5] Sidechain Ducker: 現実音バス→ -6dBまでダッキング

ウタ体験4フェーズ:
  Phase0（予感）: SE_UTA_DISTANT at -36dBFS / param_mayoi_level=0.0
  Phase1（接触）: BGM瞬断→SE_UTA_ONSET / param_mayoi_level=0.1〜0.3
  Phase2（浸食）: DSP全段適用 / param_mayoi_level=0.3〜0.8
  Phase3（頂点）: param_mayoi_level=0.8〜1.0→完全無音0.5〜1.5s→現実音復帰
```

## ボイス収録仕様（ウタ関連セリフ 専用ディレクション）

```
収録環境:

  Pitch Shift ±2〜-2cent微変動（有機感喪失）

```
