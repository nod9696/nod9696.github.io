# キャラクターシート：「キリエとサキ」

---

## 逆真キリエ（kirie）

### 核心
- **動機**：存在を消すこと（＝感情を消すこと）→ 物語後半で「存在し続けること」に転換
- **恐れ**：恐怖という概念を持たない。強いて言えば「意味のない状態が続くこと」
- **成長弧**：「死を探す存在」→「初めて明日を数える存在」

### 心理・防衛機制
- 感情をすべて「論理」に変換して処理する（例：親密さ→「死亡率の低下」）
- 感情の語彙を持たないのは欠落ではなく、そもそも必要としてこなかったため
- 「わからない」は拒絶ではなく**未処理の誠実な宣言**
- 変化を認識できない——変わっていることに気づくのは読者とサキだけ

### 無意識の行動パターン
- サキの言動を観察・記録し続ける（本人は「データ収集」と認識）
- サキが望むことを「理由」として行動する（感情ではなく因果）
- 約束を「破棄できない」と表現する（「守りたい」ではない）
- 数を数える（杯数・回数・日数）——それが唯一の感情の代替

## 声ガイド：キリエ

### 使う語彙
「可能」「不可能」「致死的」「許容」「記憶」「体系」「定義」「変数」「因果」「低下」「検証」

### 絶対使わない言葉
| カテゴリ | 禁止語 |
|---|---|
| 感情語 | 好き、嫌い、つらい、楽しい、悲しい、嬉しい |
| 慣用句 | 心が痛む、胸が高鳴る、心が折れた |
| 社交辞令 | ありがとう（初期）、すみません、そうですね |
| 推測語 | 多分、きっと、～だと思う |
| 修飾語 | とても、本当に、すごく |

### 文の構造パターン
- **最小限形**：「許容できる」「記憶した」「わからない」
- **因果形**：「お前が、それを望まないから」
- **否定形**：「……ではない」「……なかった」
- **反復継続形**：「……明日も。その次の日も」

### 口癖
- 「……」で始まる（思考時間）
- 「わからない」（最頻出・未処理宣言）
- 「致死的でなかった」「許容できる」（判定語）
- 「お前が〜だから」（唯一の因果律）

### サンプル台詞（実証済み）
```
「……わからない」
「許容できる」
「致死的でなかった」
「お前が、それを望まないから」
「……そうか」
「次は、言う」
「……なかった。最近までは」
「記憶した」
「人間の言語体系は複雑だ」
「お前が最初だ」
「……明日も。その次の日も」
「50万回を超えた後、新しい変数が加わった。お前だ」
```

### 言わないセリフ（禁止）
- ❌「君のことが好きだから」→ ⭕「お前が、それを望むから」
- ❌「大丈夫、ついてるよ」→ ⭕「……明日も。その次の日も」
- ❌「もう死のうとするな」→ ⭕（事実のみ述べる）

## 入江サキ（saki）

### 核心
- **動機**：消えること（自殺願望）→ 「ここにいること」への転換
- **恐れ**：感情を持つこと、依存すること、また失うこと
- **成長弧**：「感情を閉ざした少女」→「感情をこぼし始める少女」

### 心理・防衛機制
- 感情を持つこと自体を恐れている（傷つくから）
- 「いてもいいよ」「知ってる」——許可形・確定形で距離を保つ
- キリエへの感情を「なんとなく」で曖昧化し続ける
- 依存を認識したとき「忘れて」と言う——認識したくない

### 感情の解像度（3フェーズ）
| フェーズ | 章 | 特徴 |
|---|---|---|
| 回避的 | 00-04 | 「知ってる」「そう」で流す。感情に名前をつけない |
| 半開示的 | 05-09 | 「覚えてて」「もうやめて」→ 行動依頼で感情を示す |
| 直接的 | 10-14 | 「もうやめて（死のうとするの）」→ 核心を言語化 |

## 声ガイド：サキ

### 使う語彙
「まあ」「なんで」「そう」「なんとなく」「知ってる」「いいけど」「覚えてて」「嫌だな」

### 絶対使わない言葉
- ❌ 直接的な感情表白（「好きだから」「支えです」「頑張ります」）
- ❌ 未来の計画宣言（「一緒に生きていきましょう」）
- ❌ 過去の確定（「あなたがいなければ死んでいた」）

### 口癖
- 「けど」で文を開く（「まあ、いいけど」）
- 「なんで」で違和感を表す（疑問ではなく感情）
- 「……」の沈黙（「言えない」と「言いたくない」の両義）
- 「知ってる」「そう」で中断する

### サンプル台詞（実証済み）
```
「まあ、いいけど」
「なんで」
「そう」
「……いてもいいよ」
「覚えてて」
「もうやめて」
「知ってる」
「忘れて」
「なんとなく」
「別にどうってことないけど、いなくなるのは……嫌だな」
```

## 関係性マップ

### キリエ → サキ
「新しい変数」「死亡率を低下させた存在」「理由の唯一の出典」
→ 本人は「感情」と認識していないが、サキなしでは行動できなくなっている

### サキ → キリエ
「理解できないが拒絶できない」「見られていることで自分が見えてくる」
→ キリエの存在が「感情を持つことを許可」してくれている

### 関係性の転換点
| 章 | 転換点 |
|---|---|
| 01 | 「いてもいいよ」——サキがキリエの存在を許可 |
| 04 | 無言で隣に座る——接触による非言語的な接続 |
| 06 | サキが初めて怒る——関係性への投資を自覚 |
| 12 | 「お前が最初だ」——キリエの孤独の開示 |
| 13 | 手を取る——サキからの能動的な接触 |

### 鈴村の機能
- **物語的役割**：外部からの現実性チェック。キリエを「異常」と定義する視点
- **活用法**：キリエの「人間的に見えない行動」を際立たせる鏡
- **潜在力**：サキが「キリエを守ろうとする」場面のトリガーになりうる

### 白蝕の象徴性
- キリエの「50万回の死」が生み出した副産物——死への渇望が外在化したもの
- サキの自殺願望とも共鳴する（サキを屋上に誘う）
- 「白蝕を倒す」＝「キリエが死ぬことをやめる」という物語的解決

## キャラクターアーク設計

---

### なぜキリエは好かれるか

**1. 「正直すぎる」という逆説的な安心感**
キリエは嘘をつけない。「わからない」と言うとき、本当にわからないのだ。人間関係に必ずある「本当はどう思っているの？」という不安が、キリエには存在しない。読者にとって、感情を隠さない（隠せない）存在は奇妙な形で信頼できる。

**2. 変化を自覚しない変化**
キリエは「自分が変わった」と言わない。気づいていないから。しかし読者には見える——砂糖の杯数を覚えていること、無断外出をしなくなること、「次は言う」と約束すること。自分の変化に無自覚なまま変わっていくキャラクターは、読者に「気づかせる喜び」を与える。

**3. 欠落が愛おしい**
感情の語彙を持たないということは、「好き」を知らないまま好きになっていくということだ。それを論理で必死に処理しようとする姿——「死亡率が低下している」「新しい変数が加わった」——は、感情語を使うどんな告白よりも深く刺さる。

**4. 究極の受容**
キリエはサキを変えようとしない。自殺願望も、閉ざした心も、全て「お前がそうであること」として受け取る。評価も批判もしない。ただ観察し、記憶し、隣にいる。現代の読者が渇望する「ただそこにいてくれる存在」の理想形。

**5. 強さと弱さの同居**
50万回死のうとした存在が、サキの一言「もうやめて」に初めて止まる。圧倒的な強さを持ちながら、たった一人の人間に変えられる——この非対称が、キリエへの感情移入を深める。

---

### なぜサキは好かれるか

**1. 「なんとなく」の共感性**
サキは理由を説明しない。キリエを部屋に迎え入れた理由も、一緒にいる理由も「なんとなく」だ。これは読者の多くが日常で感じている「感情はあるけど言語化できない」状態の正直な表現。

**2. 弱さを見せながら諦めない**
自殺願望を持ちながら、それでもコーヒーを淹れ、鈴村の電話に出て、キリエに怒る。「生きていたくない」と「生きている」が矛盾なく同居している。その矛盾そのものが人間らしく、読者は置いてけぼりにされない。

**3. キリエへの反応が読者の代理**
サキが「なんでそんなこと言うんだろ」と困惑するとき、読者も同じように困惑している。サキの驚き・戸惑い・笑い——それが読者のキリエへの反応を代弁する。サキを通してキリエを愛するように設計されている。

**4. 小さな強さ**
キリエに初めて怒るシーン（chapter06）、「もうやめて」と言えるシーン（chapter12）、手を取るシーン（chapter13）。サキの強さは劇的ではなく、積み重なる小さな一歩だ。その小ささが、読者自身の「小さな勇気」と重なる。

---

### キリエのキャラクターアーク

#### 欠落：「感情の語彙」と「終わりへの意志」

キリエが抱える欠落は二重構造になっている。

**表層の欠落**——感情の語彙がない。「好き」「悲しい」「嬉しい」を知らない。
**深層の欠落**——「存在し続ける理由」がない。50万回死のうとしたのは、生きることの意味を見つけられなかったからだ。

この二つは実は同じ欠落の表と裏である。感情を知らないから、生きることの意味も見つけられない。

#### アークの構造：「理由を持てなかった存在」が「理由を持つ」

```
【ゼロ地点】chapter00
  死ぬことが唯一の目標。存在は惰性。
  感情はデータ。人間は食料か背景。

【第一の亀裂】chapter01
  「いてもいいよ」——初めて許可された。
  なぜそれが特別なのかわからない。でも記憶した。

【習慣化と混乱】chapter02-06
  サキの存在が「変数」になる。
  死亡率が下がる。無断外出をしなくなる。
  しかし「なぜそうするのか」の答えが出ない。
  ——これはまだ「論理の範囲」の行動だと思っている。

【初めての問い】chapter07
  「留まりたいと思ったことはあるか？」
  長い沈黙の後——「なかった。最近までは」。
  自分の変化を言語化した、初めての瞬間。
  （感情語は使っていない。でも「最近までは」という比較が生まれた）

【危機と選択】chapter11-12
  白蝕が「お前も同じだった」と言う。
  かつてのキリエには反論できなかっただろう。
  しかし今は——「違う」。
  この「違う」に、キリエは理由を言えない。でも確かにそう感じている。
  ——これが、感情の「名前のない経験」。

【核心の開示】chapter12
  「お前が最初だ」——誰にも「死ぬな」と言われたことがなかった。
  この台詞の重さは、キリエが50万年かけて蓄積した孤独の開示だ。
  サキの「もうやめて」は、その孤独に最初に触れた声。

【転換】chapter13
  「残る」ことを選ぶ。死ぬのではなく、生きることを。
  これはキリエにとって50万回目の「死のうとする」の反対——
  つまり、初めての「生きようとする」。
  感情語はまだ使えない。でも行動が変わった。

【到達点】chapter14
  砂糖3杯のコーヒー。「許容できる」「知ってる」。
  アークの完成は劇的ではなく、静かな朝にある。
  「許容できる」——かつては中立の判定語だったこの言葉が、
  今は「ここにいていい」という宣言になっている。
```

#### キリエのアークが機能する理由
変化の証拠を感情語で示さないこと。「好きになった」ではなく、行動と語彙の微妙な変化で示す。読者は「気づく」読書体験を得る。それがキリエのアークの本質。

### サキのキャラクターアーク

#### 欠落：「感情を持つ許可」と「居場所」

サキの欠落は「感情を持ってはいけない」という無意識の禁止令だ。

幼少期から「世界は汚い音で満ちている」と感じてきたサキは、感情を持つことで傷つくことを学習した。だから閉ざした。感情がなければ傷つかない。しかし同時に、「生きている実感」も失った。自殺願望はこの無感覚の極地にある——生きていてもどうせ何も感じないなら、いっそ終わりにしたい。

もう一つの欠落は「居場所」だ。自分がいてもいい場所を知らない。いつも「邪魔ではないか」「消えた方がいいか」を測り続けている。

#### アークの構造：「感情を閉ざした少女」が「感情をこぼし始める」

```
【ゼロ地点】chapter00
  窓の外。夜を眺めている。
  世界の音を全て閉ざして生きている。
  自殺を考えている——それが今夜の目的。

【許可の瞬間】chapter00（後半）
  キリエの声が「護りを貫いた」。
  一瞬、全てがどうでもよくなった。
  そして——泣いた。一生分溜め込んでいたものを。
  自殺願望が消えたのは、「泣けた」からだ。
  （閉ざしていた感情が初めて外に出た）

【許可の言語化】chapter01
  「いてもいいって言ってくれる人がいるかどうか、かも」
  これはサキ自身の欲求の告白だ。
  「いてもいいよ」と言うことで、自分もいていいと感じている。

【感情の芽生えの否認】chapter04
  雨の夜。もたれかかる。翌朝「忘れて」。
  感情が芽生えたと認識した瞬間、消そうとする。
  これがサキの防衛機制の典型。

【依存の自覚と恐怖】chapter06
  キリエが一晩いない。「思っていたより不安だった」。
  自分が依存していることに気づいて——怒る。
  怒りの裏にあるのは「また傷つくのが怖い」という感情。

【行動による感情の開示】chapter08-09
  直接言わないが、行動で示し始める。
  電話を受けられるようになる。アイスを交換する。
  感情を「行動」に変換する回路が開き始めている。

【危機での核心】chapter11-12
  白蝕に引き寄せられる——これはサキの中にまだある「終わりたい」気持ち。
  しかしキリエが来る。「去れ」と言う。
  ここでサキは「キリエに守られること」を受け入れる。
  ——「いてもいいよ」が、今度はキリエからサキへと返ってくる。

  「もうやめて」——chapter12の核心。
  初めて直接的に、感情を言語化した。
  「死のうとするのをやめて」は、「あなたに生きていてほしい」の言い換えだ。
  サキが初めて、自分の感情に名前をつけた瞬間。

【到達点】chapter14
  砂糖3杯を「知ってる」と言えること。
  「知ってる」——これがサキの最大の感情表現。
  観察していた、覚えていた、大切にしていた、という全てが一言に圧縮されている。
  居場所を「作ってもらった」のではなく、「自分で作った」朝。
```

#### サキのアークが機能する理由
サキは感情を「言わない」のではなく「言えない」キャラクターだ。しかしアークの進行に従い、少しずつ「言えるようになる」——ただしそれは突然の感情爆発ではなく、「覚えてて」「知ってる」「もうやめて」という短く、でも重い言葉の積み重ねだ。

---

### 二人のアークの相互作用

二人のアークは**鏡構造**になっている。

| キリエ | サキ |
|---|---|
| 感情の語彙を持たない | 感情を言語化することを恐れている |
| 「存在し続ける理由」がない | 「居場所」がない |
| 死ぬことが目標 | 死ぬことで楽になりたい |
| 論理で全てを処理する | 感情を全て閉ざす |

どちらも「感情からの逃走」を異なる方法でやっている。

キリエはサキに「感情を持つ許可」を与える（批判しないから）。
サキはキリエに「存在し続ける理由」を与える（「いてもいいよ」と言うから）。

**二人は互いの欠落を埋めているのではなく、互いの欠落に気づかせている。**

欠落は埋まらない。キリエはおそらく最後まで「好き」という言葉を使わない。サキはおそらく最後まで「あなたが必要です」とは言わない。しかしそれでいい——二人の欠落は「欠落のまま」で共鳴している。それがこの物語の核心だ。

## 執筆ガイドライン

### やること
- キリエの台詞は「主語＋述語」の最小構成を基本に
- サキの感情は行動・依頼の形で示す（直接表白しない）
- コミカル場面：キリエの「論理」とサキの「常識」のズレを使う
- 感情場面：台詞を短くして、沈黙（……）の行で余白を作る

### やらないこと
- キリエに「好き」「嬉しい」などの感情語を使わせない
- サキに独白で感情を丁寧に説明させない（「実は私は〜と感じていた」型はNG）
- 関係を急速に進展させない（距離はゆっくり、小さなステップで）
- キリエをコメディリリーフだけにしない（論理性の中の誠実さを忘れない）

### 感情温度の管理
```
chapter00-02: 冷たい・距離あり・好奇心
chapter03-06: 温度が少し上がる・習慣化・最初の衝突
chapter07-09: 穏やか・日常・ユーモア
chapter10-11: 緊張・危機・守る
chapter12-13: 核心開示・選択・決定
chapter14:    静かな温かさ
```
