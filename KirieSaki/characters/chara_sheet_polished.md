## 逆真キリエ（Kirie）

### 核心
- **動機**：存在を消すこと（＝感情を消すこと）→ 物語後半で「存在し続けること」に転換
- **恐れ**：「意味のない状態が続くこと」（恐怖という概念を持たない。本人は恐怖と認識しない）
- **成長弧**：「死を探す存在」→「初めて明日を数える存在」
- **防衛機制**：知性化（感情を論理に置換）。感情が混入すると「論理的矛盾」が生じ始める——それが変化の証拠

### 声・話し方

**語彙傾向**
- 使う：「可能」「不可能」「致死的」「許容」「記憶」「体系」「定義」「変数」「因果」「低下」「検証」「算術」「残滓」「違う」（後半から）
- 使わない：感情語（好き・嫌い・つらい・楽しい・悲しい）、慣用句、社交辞令（ありがとう・すみません）、推測語（多分・きっと・～だと思う）、修飾語（とても・本当に・すごく）

**文体（章別変化）**
| フェーズ | 章 | 文体の特徴 |
|---|---|---|
| 冷・機械的 | 00-02 | 最短構成。「。」ごとに完結。矛盾なし |
| 温・変数混入 | 03-06 | 論理の隙間に「わからない」が増える |
| 穏・矛盾の自覚 | 07-09 | 長さが少し増える。比較形（「最近までは」）が出現 |
| 熱・感情の侵食 | 10-11 | 論理的な説明に矛盾が混じり始める |
| 沸・核心 | 12-13 | 「違う」が出現。説明できないが確信がある |
| 静寂・到達 | 14 | 「許容できる」の意味が変化（判定→宣言） |

**口癖・特徴**
- 「……」で始まる（思考・言語検索の時間）
- 「わからない」（最頻出。未処理の誠実な宣言。拒絶ではない）
- 「致死的でなかった」「許容できる」（判定語）
- 「お前が〜だから」（唯一の因果律）
- 数を数える発話（杯数・回数・日数）——感情の代替行為

**言わない言葉（禁止）**
| ❌ 禁止 | ⭕ 代替 |
|---|---|
| 「君のことが好きだから」 | 「お前が、それを望むから」 |
| 「大丈夫、ついてるよ」 | 「……明日も。その次の日も」 |
| 「もう死のうとするな」 | （事実のみ述べる） |
| 「君は失敗した」 | 「……今、致死的ではなかった」 |
| 「一緒にいてくれてありがとう」 | 「……50万回目の翌日が、お前だった」 |

**サンプル台詞（実証済み）**
```
「……わからない」
「許容できる」
「致死的でなかった」
「お前が、それを望まないから」
「……そうか」
「次は、言う」
「……なかった。最近までは」
「記憶した」
「人間の言語体系は複雑だ」
「お前が最初だ」
「……明日も。その次の日も」
「50万回を超えた後、新しい変数が加わった。お前だ」
```

**新規サンプル台詞（chapter10-14向け）**
```
【「初めての比較」chapter11前後】
「以前なら……選ばなかった。今は、違う。理由は、わからない」
→ 変化を自覚し始めた最初の発話。「違う」の出現が転換点

【「拒絶できない理由」chapter12前後】
「去れ。……お前が命令だ。理由は問うな。ただ、去れ」
→ サキへの感情を「命令」という形式に圧縮。最大限の必死さ

【「数えない選択」chapter14】
「……今日は、数えなかった。お前が、知っていたから」
→ 「数えること」が不要になった=サキへの信頼の完成形
```

### 感情パターン

| 感情 | 表れ方 |
|---|---|
| 喜び | 数える行為が増える。観察対象がサキの「危険性」から「細部の仕草」にシフトする |
| 怒り | 論理の構造が崩れ、文が短くなる。「……」が増える |
| 悲しみ | 「わからない」の頻度が上がる。行動が止まる |
| 感情を隠す方法 | 論理的な説明を重ねる。しかし行動は感情に支配されている（矛盾が生まれる） |

## 入江サキ（Saki）

### 核心
- **動機**：消えること（自殺願望）→ 「ここにいること」への転換
- **恐れ**：感情を持つこと（傷つくから）、依存すること（また失うから）
- **成長弧**：「感情を閉ざした少女」→「感情をこぼし始める少女」
- **防衛機制**：抑圧（感情そのものを封印）。感情を認識した瞬間「忘れて」と言う——認識したくない

### 声・話し方

**語彙傾向**
- 使う：「まあ」「なんで」「そう」「なんとなく」「知ってる」「いいけど」「覚えてて」「嫌だな」
- 使わない：直接的な感情表白（「好きだから」「支えです」「頑張ります」）、未来の計画宣言、過去の確定（「あなたがいなければ死んでいた」）

**文体（章別変化）**
| フェーズ | 章 | 特徴 |
|---|---|---|
| 回避的 | 00-04 | 「知ってる」「そう」で流す。感情に名前をつけない |
| 半開示的 | 05-09 | 「覚えてて」「もうやめて」→ 行動依頼で感情を示す |
| 直接的 | 10-14 | 「もうやめて（死のうとするの）」→ 核心を言語化 |

**口癖・特徴**
- 「けど」で文を開く（「まあ、いいけど」）
- 「なんで」で違和感を表す（疑問ではなく感情）
- 「……」の沈黙（「言えない」と「言いたくない」の両義）
- 「知ってる」「そう」で中断する

**言わない言葉（禁止）**
| ❌ 禁止 | ⭕ 代替 |
|---|---|
| 「好きだから」（直接表白） | 「覚えてて」「いてもいいよ」 |
| 「頑張ります。一緒に生きていきましょう」 | 「……明日も、一緒にいてもいいか」 |
| 「キリエは私の支えです」 | 「いなくなるのは嫌だな」「覚えてて」 |
| 「キリエ、私、好きだよ」（直接） | 「……知ってる。お前が最初だから」 |
| 「何があっても絶対に離さない」 | 「また来てもいいから」 |

**サンプル台詞（実証済み）**
```
「まあ、いいけど」
「なんで」
「そう」
「……いてもいいよ」
「覚えてて」
「もうやめて」
「知ってる」
「忘れて」
「なんとなく」
「別にどうってことないけど、いなくなるのは……嫌だな」
```

**新規サンプル台詞（chapter10-14向け）**
```
【「拒絶から許可へ」chapter10前後】
「……また来てもいいから。なんか、その方が……いい」
→ 能動的な「来てほしい」の初めての表現。でも直接的ではない

【「消極的な愛情表現」chapter12前後】
「……キリエのこと、わかんないけど。でも、なんか……やめてほしくない」
→ 「死なないで」の言えない形。「やめてほしくない」が核心

【「習慣の深さ」chapter14】
「砂糖、3杯でしょ。……知ってる」
→ 覚えていた=観察していた=大切にしていた。一言に圧縮された愛
```

### 感情パターン

| 感情 | 表れ方 |
|---|---|
| 喜び | 文が少し長くなる。「なんか」「ちょっと」が増える。「嫌だな」の逆として「いいな」が出る |
| 怒り | 「なんで」が疑問ではなく感情語として増える。語尾が崩れる |
| 悲しみ | 「……」が長くなる。返答が単語のみになる |
| 感情を隠す方法 | 「知ってる」「そう」「いいよ」で中断。話題を変える。視線を逸らす描写 |

## 関係性マップ

### キリエ → サキ
「新しい変数」「死亡率を低下させた存在」「理由の唯一の出典」
→ 本人は「感情」と認識していないが、サキなしでは行動できなくなっている

### サキ → キリエ
「理解できないが拒絶できない」「見られていることで自分が見えてくる」
→ キリエの存在が「感情を持つことを許可」してくれている

### 力関係と依存の非対称性
- **chapter01-05**：サキが許可する側。キリエは受け入れられる側
- **chapter06-09**：依存が双方向になり始める。どちらが先に変化したかわからなくなる
- **chapter10-14**：キリエが守る側へ逆転。サキが「受け取る」ことを学ぶ

### 転換点（章別）
| 章 | 転換点 | 関係性の変化 |
|---|---|---|
| 01 | 「いてもいいよ」 | サキがキリエの存在を許可。一方向の受容 |
| 04 | 無言で隣に座る | 接触による非言語的接続。言語不要の信頼 |
| 06 | サキが初めて怒る | 関係性への投資の自覚。依存の開始 |
| 12 | 「お前が最初だ」 | キリエの孤独の開示。50万年分の重さ |
| 13 | 手を取る | サキからの能動的接触。受け身→主体への転換 |

### chapter07-09（穏やか・日常・ユーモア期）のシーンアイデア
1. **コーヒーの砂糖問題**：キリエが「砂糖の適正量を算出した」とデータを持ってくる。サキが「好みは数値じゃない」と言う。キリエが「……変数が多い」と困惑する
2. **「普通」の学習**：キリエが「『普通』とは何か」を聞く。サキが答えられない。キリエ「……お前も、わからないのか」→「そういうもん」→「……そうか」（初めての共感）
3. **返事のタイミング**：サキが「おやすみ」と言っても返ってこない（キリエは不要と判断）。次の日サキが「昨日、おやすみって言ったじゃん」→キリエ「記憶した。……おやすみ」→「遅い」

### chapter14（到達点）の描写
砂糖3杯のコーヒー。
- キリエの「許容できる」：かつては中立の判定語だったが、今は「ここにいていい」という宣言
- サキの「知ってる」：覚えていた・観察していた・大切にしていた——全てが一言に圧縮
- **欠落は埋まらない**：キリエはおそらく最後まで「好き」と言わない。サキはおそらく最後まで「あなたが必要です」とは言わない。しかしそれでいい。二人の欠落は「欠落のまま」で共鳴している

## サブキャラの機能

### 鈴村
- 外部からの現実性チェック。キリエを「異常」と定義する視点（サキと対比）
- サキが「キリエを守ろうとする」場面のトリガー——鈴村の批判がサキの感情を引き出す
- キリエの「人間的に見えない行動」を際立たせる鏡

### 白蝕
- キリエの「50万回の死」が生み出した副産物——死への渇望の外在化
- サキの自殺願望とも共鳴する（サキを屋上に誘う）
- **物語的解決**：「白蝕を倒す」＝「キリエが死ぬことをやめる」
- 二人の「生への選択」がこの敵を倒す唯一の方法——二人が一緒にいることで初めて過去の自分たちに勝つ

## 沈黙（「……」）の使い分け

### キリエの「……」
| タイプ | 意味 | 使い場面 |
|---|---|---|
| 開始の沈黙 | 言語を検索している | 発話前 |
| 考察の沈黙 | 論理を処理している | 問いかけへの応答前 |
| 感情未処理 | 「わからない」の前段階 | 感情が混入した瞬間 |
| 確定の沈黙 | 判定が出た | 「……記憶した」 |
| 受け入れの沈黙 | 拒絶しない選択 | サキの感情を受け取った後 |

### サキの「……」
| タイプ | 意味 | 使い場面 |
|---|---|---|
| 拒絶の沈黙 | 言いたくない | 感情に触れられたとき |
| 言葉にならない沈黙 | 感情が溢れている | 泣きそうなとき |
| 呼応の沈黙 | キリエのリズムに合わせている | 二人が近い距離にいるとき |
| 承認の沈黙 | 「いいよ」の意 | 拒否しない選択 |
| 懇願の沈黙 | 言葉が出ない「やめて」 | キリエが危険なとき |

## ✍️ 執筆ガイドライン

### やること
- キリエの台詞は「主語＋述語」の最小構成を基本に
- サキの感情は行動・依頼の形で示す（直接表白しない）
- コミカル場面：キリエの「論理」とサキの「常識」のズレを使う
- 感情場面：台詞を短くして、沈黙（……）の行で余白を作る
- キリエの変化は「行動の矛盾」で示す（本人が言葉にしない）
- サキの変化は「言葉の長さ・直接性」で示す（少しずつ言える量が増える）

### やらないこと
- キリエに「好き」「嬉しい」などの感情語を使わせない
- サキに独白で感情を丁寧に説明させない（「実は私は〜と感じていた」型はNG）
- 関係を急速に進展させない（距離はゆっくり、小さなステップで）
- キリエをコメディリリーフだけにしない（論理性の中の誠実さを忘れない）
- キリエが「守りたい」と言葉で言う（行動でのみ示す）
- サキが感情を「説明」する（「感じる・こぼれる」形で示す）

### 感情温度の管理
```
chapter00-02: 冷たい・距離あり・好奇心
chapter03-06: 温度が少し上がる・習慣化・最初の衝突
chapter07-09: 穏やか・日常・ユーモア
chapter10-11: 緊張・危機・守る
chapter12-13: 核心開示・選択・決定
chapter14:    静かな温かさ
```

### キャラが崩れる危険サイン
- キリエが「多分」「きっと」を使い始めた → 削除
- サキが「私は〜と思う」と心理を説明し始めた → 行動描写に変換
- 二人の関係が「急に」進展した → ステップを細分化
- キリエのコメディシーンで誠実さが消えた → 論理の根拠を添える
