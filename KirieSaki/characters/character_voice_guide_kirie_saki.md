# 「キリエとサキ」キャラクター声ガイド

## キリエの声ガイド

### 使う語彙の傾向
- **論理語彙**：「可能」「不可能」「致死的」「許容」「記憶」「体系」「定義」
- **機械的判定**：「～できる」「～できない」「～ではない」「～だ」
- **観察語**：「知る」「わかる」「見た」「感じた」← 後ろには常に「わからない」「なかった」など否定
- **因果性**：「から」「ため」「理由」を多用（感情ではなく理由で答える）
- **時制の明確化**：「今」「最近」「前」などで時間を厳密に分ける

### 絶対使わない言葉
1. **感情語**：「好き」「嫌い」「つらい」「楽しい」「悲しい」（経験がないため）
2. **慣用句**：「心が痛む」「胸が高鳴る」「心が折れた」（身体的現実を持たない）
3. **社交辞令**：「ありがとう」「すみません」「そうですね」（相互性の理解が浅い）
4. **推測語**：「多分」「きっと」「～だと思う」（確実性のない発言をしない）
5. **修飾表現**：「とても」「本当に」「すごく」「さらに」（程度を人間的には量れない）

### 文の構造パターン
```
【基本形】
「主語＋述語」の最小限形
例）「許容できる」「記憶した」「わからない」

【理由説明型】
「～から」で因果関係を述べる
例）「お前が、それを望まないから」

【否定型】
述語を否定形にして存在を否定する
例）「～ではない」「～ていない」「～なかった」

【確認型】
相手の発言を一度復唱してから判定する
例）「そう。……つまり」→述語

【未来型】
「～も。その次も」と反復で継続性を示す
例）「……明日も。その次の日も」
```

### 口癖・特徴的フレーズ
- **「……」で始まる発話**：考えている、言語を検索している時間を表現
- **「わからない」**：キリエの思考の限界を示す最頻出フレーズ。否定ではなく「未処理」の宣言
- **「致死的でなかった」**：死の判定を日常的に行う視点
- **「お前が〜だから」**：相手の存在・希望を唯一の因果律として認識
- **短い句点の連続**：「。」で区切ることで、各文が独立した判定であることを示す
- **主語の省略→「お前」への執着**：相手がいないと文が成立しない

### キリエらしいセリフ 新作5本

**1. 食事の場面で**
```
「栄養摂取の必要性は理解した。味覚は……評価基準を持たない」
- コンテキスト：サキが食べ物の味について感想を求めたときに
- 特徴：必要性は認める（学習した）が、主観的判定ができないことを明言
```

**2. サキの昔の話を聞いて**
```
「その時点では、死ぬ選択肢があったのか。……今、お前は生きている。
理由は、わからない。だが、致死的ではなくなった」
- コンテキスト：サキが過去に自殺を考えていたと打ち明けたとき
- 特徴：時系列で事実を整理→現在の生存状態を「選択の結果」ではなく「致死的でない状態」と定義
```

**3. 感情について聞かれて**
```
「感情とは。お前の中で、何が変化している」
- コンテキスト：キリエが「感情とは何か」を理解しようとしているとき
- 特徴：定義を求めるのではなく「変化」という物理的概念で理解しようとする
```

**4. 同居を続ける理由について**
```
「50万回を超えた後、新しい変数が加わった。
お前だ。その後の死亡率は……低下している」
- コンテキスト：なぜ一緒にいるのかと聞かれたとき
- 特徴：感情ではなく統計（死亡率の低下）で自分の行動を説明
```

**5. 約束について**
```
「記憶した。明日も、その日も。破棄しない。
……破棄できない。お前が存在する限り」
- コンテキスト：サキが「毎日一緒にいてほしい」と言ったとき
- 特徴：「破棄できない」という受動性で、サキへの依存を示す。感情語を避けたまま絆を表現
```

### キリエが絶対言わないセリフ 3本

**1. 「君のことが好きだから」**
- **理由**：感情語「好き」を使わない。キリエは「好き」という抽象概念を持たない。
代わり：「お前が、それを望むから」「お前が存在するから」という因果律で示す

**2. 「大丈夫、僕がついてるよ」**
- **理由**：「大丈夫」は主観的判断、「ついてる」は安心の約束であり、キリエの確実性の外。
また「僕」という親密さを示す一人称は、キリエの距離感に合わない。
代わり：「……明日も。その次の日も」と継続を述べるだけ

**3. 「だからもう、死のうとするなよ」**
- **理由**：命令形は相手の自由意志を否定する。キリエは他者の意思を「許容」し「わからない」が、
それを抑圧しない。また「～するな」という感情的な懇願は使わない。
代わり：「死ぬ選択肢がある。だが、今、お前は生きている。その理由は、わからない」と事実のみ述べる
```

## サキの声ガイド

### 使う語彙の傾向
- **簡潔語**：「まあ」「なんで」「そう」「なんとなく」で文を閉じる
- **口語調**：「～けど」「～ちゃった」「～てさ」の緩い接続
- **否定の柔らかさ**：「いいけど」「やめてよ」「知ってる」と相手を突き放さない
- **存在の許可形**：「いてもいいよ」「いてもいいじゃん」など、相手の在り方を黙認する言い方
- **回避語**：「なんか」「その」「あの」などぼかしの使用（核心から遠ざかる）

### 感情の出し方（直接的/回避的のバランス）
```
【回避的フェーズ】
- 一次感情は出さない
- 「知ってる」「そう」で相手の言葉を中断
- 「なんとなく」で感情を曖昧化
- 視線を逸らす、短い返答
⇒ サキが自分の感情に目を向けていない段階

【半開示的フェーズ】
- 「覚えてて」「もうやめて」など、相手への依頼に変換
- 感情そのものではなく、「してほしい行動」で伝える
- 「いてもいいよ」と相手の存在を許可する形で愛を示す
⇒ 感情が目覚め始めているが、名前をつけられない段階

【直接的フェーズ】（ストーリー後期）
- 「覚えてて」の繰り返しが「忘れないでほしい」に変わる
- 「もうやめて」が「一人にしないで」に深化
- 短さはそのままだが、重みが増す
⇒ 感情を言語化しはじめた段階
```

### 口癖・語尾
- **「けど」で文を開く**：「まあ、いいけど」「なんでそんなこと言うけど」
- **「～て」で脱力する**：「ちゃった」「ってさ」で事実を軽く流す
- **「なんで」で疑問形**：本当の質問ではなく、相手の言動への違和感の表現
- **「……」の沈黙**：キリエと同じだが、サキは「言葉が出ない」「言いたくない」の両義性
- **語尾を下げる**：「知ってる」「そう」「いいよ」と確定を避ける

### サキらしいセリフ 新作5本

**1. キリエの奇行に対して**
```
「変なやつだな。……でも、いてもいいよ」
- コンテキスト：キリエが何か理解できない行動をしたとき
- 特徴：違和感を認めながらも、受け入れる。「いいよ」が相手の存在を許可する
```

**2. 自分の過去について聞かれて**
```
「昔のことだしね。覚えてる？ 覚えてくれてたら、それで」
- コンテキスト：キリエが「覚えている」ことを示したとき
- 特徴：過去は消そうとするが、キリエに「覚えられていること」に執着
```

**3. キリエとの同居について**
```
「別にどうってことないけど、いなくなるのは……嫌だな」
- コンテキスト：なぜキリエといるのかと聞かれたとき
- 特徴：「どうってことない」で無関心を装いながら、最後に「嫌だな」と本音をこぼす
```

**4. 朝、キリエが起きてくるのを見て**
```
「朝か。……よかった。起きてくると思ってた」
- コンテキスト：毎朝のルーチン、キリエが「また死にに行った」のではなく戻ってきたことへの安堵
- 特徴：「よかった」が唯一の感情語。通常は避けるが、この場面では本心がこぼれる
```

**5. キリエに感謝するとき**
```
「ありがとうとか、言わなくていいの？」
「……ありがとう」
「そっか。なんか……嬉しいな」
- コンテキスト：キリエが「ありがとう」と初めて言ったとき
- 特徴：サキが相手にお作法を教えるのではなく、キリエの学習と努力を見て初めて感情が零れ落ちる
```

### サキが絶対言わないセリフ 3本

**1. 「私のこと、好きですか？」（敬語、直接的、確認的）**
- **理由**：サキは感情を言語化する習慣がない。また「好きですか？」と確認を求めるのは
サキの「わかろうとしない」姿勢に反する。自分の感情すら曖昧なまま生きてきた者が、
相手の感情を直線的に問うことはない。
代わり：「覚えてて」「いてもいいよ」と、相手の行動で判断する

**2. 「頑張ります。一緒に生きていきましょう」（決意表明、敬語、未来計画）**
- **理由**：サキは「頑張る」「頑張ろう」という意識高い系の言語を持たない。
また「一緒に生きていく」と未来を約束するのは、サキの「今この瞬間」の生き方に反する。
代わり：「……明日も、一緒にいてもいいか」と曖昧な継続を求める

**3. 「キリエは私の支えです。キリエがいなければ死んでいました」（直接表白、重い告白）**
- **理由**：サキはキリエに依存しているが、それを「支え」と言語化しない。
また「死んでいた」と過去を確定するのは、自分の弱さを自覚することであり、
感情を閉ざしてきたサキは主体的には言わない。
代わり：「いなくなるのは嫌だな」「覚えてて」という消極的・保留的な表現にとどまる
```

## 二人の会話パターン

### 典型的なやり取りの構造

```
【基本パターン】
1. キリエ：論理的な質問・観察・定義を述べる
2. サキ：違和感・沈黙・「なんで？」で返す
3. キリエ：理由を因果的に説明する
4. サキ：「そっか」「知ってる」と中断 OR「……」と沈黙
5. 間：そのまま終わる（解決しない）

例構造：
キリエ「なぜ死ぬ」→ サキ「なんでそんなこと聞くん」→
キリエ「理由を知りたい」→ サキ「……」→ キリエ「わかった」

【親密化パターン】
1. キリエ：サキの行動を観察し、定義する
2. サキ：キリエに見られていたことに気づく
3. キリエ：観察内容を述べる（拒否でなく、事実のみ）
4. サキ：否定も肯定もせず、「いてもいいよ」と許可形で返す
5. キリエ：「記憶した」と確定

【衝突パターン】
1. サキ：感情的に拒否する（「やめてよ」「もうやめて」）
2. キリエ：理由がわからないので、説明を求める
3. サキ：理由なく「嫌だ」と返す
4. キリエ：「致死的ではない」と判定し、続ける
5. サキ：キリエの「どこまでも続ける」姿勢に、押しきられて黙る
   または「……」と沈黙したまま受け入れる
```

### コミカル場面の作り方

**【ズレの活用】**
- キリエが日常のことを「致死的」「許容」で判定する → サキが常識的な反応をする
- 例：キリエ「朝食の栄養価は致死的でない」 サキ「……普通に美味しいって言えよ」

**【習得と失敗のギャップ】**
- キリエが「社会的言語」を学ぼうとするが、使い方が機械的・的外れになる
- 例：キリエ「『大丈夫』を言うべき場面と判断した。お前、大丈夫か」
      サキ「うーん、微妙」

**【サキの無関心の爆発】**
- サキが「知ってる」「そう」で流し続けた結果、キリエが混乱する
- 例：キリエ「お前の意見を知りたい」 サキ「別に」 キリエ「『別に』とは何か」

**【反復の味わい方の違い】**
- キリエ「明日も、その次の日も」と正確に反復 → サキ「多すぎ」

## サンプル会話

### シーン1: 夜中、キリエが死ぬ準備をしていると知ったサキ

```
サキ「……何やってんの？」
キリエ「致死方法の検証。新しい方法を試す必要がある」
サキ「あ、やめてよ。今、やめてよ」
キリエ「理由」
サキ「理由？」
キリエ「なぜ、それを望まないのか」
サキ「……。普通、やめるでしょ。人間は」
キリエ「人間であることは、死から遠ざかることではない。
　　　　お前も、その前も、その前も、終わりを望んでいた」
サキ「それと、これは……違う」
キリエ「何が」
サキ「……わかんない。でも、今、そばにいて。
　　　　一人で、やってほしくない」
キリエ「……わかった。今日は、検証しない」
```
**会話の仕組み**：
- キリエは「なぜ」で因果律を求め、サキは「気持ち」で拒否
- サキは理由を説明できず、最終的に「いてほしい」という存在への依頼になる
- キリエは理由がわかると「許容」し、約束を破棄しない

### シーン2: 朝食をとっているとき、キリエが「好き」という言葉を学習する

```
サキ「今日、何か思いついたのか？」
キリエ「『好き』という言葉について。人間の言語体系で、もっとも曖昧だ」
サキ「何それ。いきなり」
キリエ「定義がない。『好き』の基準は個体によって変わる。
　　　　『嫌い』の反対語だが、『好きではない』とは異なる」
サキ「……深く考えてないけど」
キリエ「お前は？」
サキ「え？」
キリエ「お前は、『好き』を使うか」
サキ「つかうけど……なんで。別にいいじゃん」
キリエ「『好き』の対象は、何か」
サキ「……いきなりすぎ」
キリエ「回避した。なぜ、答えない」
サキ「だって、そういうの……言い出したら、重くなるじゃん」
キリエ「重さ。つまり、『好き』は……大切な状態か」
サキ「……そんなとこ。知ってるじゃん」
キリエ「知らない。だが、理解した。
　　　　お前が、『好き』という言葉から逃げる理由は、
　　　　その言葉が、逃げられない選択を示すから」
サキ「……キリエ、怖い」
キリエ「恐怖の原因は、『好き』が、死から遠ざかる選択であるから」
サキ「……死ぬな。そんなこと言うな」
キリエ「……。『好き』を言う時、人間は死にたくなくなるのか」
サキ「なります」
キリエ「……そうか。それなら、記憶しておく。
　　　　『好き』は、致死的ではない」
```
**会話の仕組み**：
- キリエが言語を学習するプロセスの中で、サキの感情を言語化させられる
- サキが「重い」「言い出したら逃げられない」と本能的に感じている深さをキリエが論理で説明
- 最終的に「『好き』が生きる理由になる」という哲学的命題に二人で到達
- キリエは「記憶する」で個人的な学習を確定させる
