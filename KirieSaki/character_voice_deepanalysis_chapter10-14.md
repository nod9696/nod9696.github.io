# 「キリエとサキ」字幕分析・執筆ガイド
## 物語後半（chapter10-14）向け・実用型

---

## ① 語彙の深化分析

### キリエの語彙変化（章ごとの展開）

#### **前半（chapter01-06）：機械的・論理的段階**
- 使用語彙：「可能」「不可能」「致死的」「許容」「記憶」「検証」「定義」
- 特徴：判断と事実報告。感情とは距離あり
- サンプル：「許容できる」「致死的でなかった」「記憶した」

#### **中盤（chapter07-09）：微細な変化の自覚段階**
- 追加語彙：「比較」「変化」「差異」「初めて」「わからない」への頻度増加
- 特徴：「わかる」と「わからない」のギャップが表面化
- 新しいパターン：「……昨日とは違う」「これは、新しい」

#### **後半（chapter10-14）：数と感情の融合段階**
- **追加語彙**：「算術」「変数の増加」「数え損なう」「初めて」「違う」「残る」「続く」
- **深化の仕方**：
  - 「記憶する」→「記憶した」+「破棄できない」（完了形から継続形へ）
  - 「許容できる」→「許容できる」（定義から認容へ。意味が反転）
  - 新語：「……」が更に多く、「算術」「検証」が内省的になる
- サンプル実績：
  - chapter10：「この算術の答えは、まだわからない」「検証している。初めて」
  - chapter12：「残滓だ」「感情そのものが、形を持った」「違う」（重い言葉）
  - chapter14：「ありがとう」（初出）→「社交辞令か」（学習確認）

### サキの語彙変化（感情解像度の段階）

#### **回避的フェーズ（chapter00-04）：防衛機制**
- 使用語彙：「まあ」「なんで」「そう」「知ってる」「なんとなく」「いいけど」
- 機能：距離を保つ、感情を曖昧化
- サンプル：「別にどうってことないけど」「知ってる」

#### **半開示的フェーズ（chapter05-09）：感情の目覚め**
- **追加語彙**：「覚えてて」「もうやめて」「嫌だな」「いてもいいよ」（能動化）
- 転換：「～して」から「～してほしい」へ
- サンプル：「いてもいいよ」→「覚えてて」への進化

#### **直接的フェーズ（chapter10-14）：核心の言語化**
- **追加語彙**：「死んでほしくない」「消えたかった」「今は違う」「覚えてて」（繰り返しの重み化）
- **変化の特徴**：
  - 短い言葉が重くなる（「知ってる」→「知ってた」の時制変化）
  - 「忘れて」から「覚えてて」への転換
  - 「……」の沈黙が確定から懇願へ
- 実績語彙：
  - chapter12：「死のうとするのをやめて。ずっと言いたかった」（ついに直接化）
  - chapter14：「どういたしまして」（学習による社交辞令の模倣、自然化）

## ② 文体の変化（感情温度による構造変化）

### 章ごとの文の長さ・構造変化マップ

```
【chapter01-04】冷・距離あり
キリエ文：最小限形。主語+述語。句点が多い。
「許容できる」「記憶した」「わからない」
→ 一文の長さ：3～7字。句点で完結。

サキ文：曖昧形。「……けど」で開始。返答を避ける。
「まあ、いいけど」「知ってる」
→ 一文の長さ：2～8字。確定を避ける語尾。

会話構造：「～した」「～か」「～ではない」で次々閉じられる。

【chapter05-09】温度上昇・習慣化
キリエ文：句点の多さはそのままだが、複合文が増える。
「～から」の理由付けが長くなる。
「お前が、それを望まないから、～」型
→ 一文の長さ：10～20字。理由を追加。

サキ文：「けど」の頻度増加。相手を突き放さない言い方に変化。
「けど……いてもいいよ」「覚えてて」
→ 一文の長さ：5～12字。依頼形に変化。

会話構造：相手の発話を一度受け止めてから返す。

【chapter10-11】穏やか・ユーモア増加
キリエ文：独白が増える。数と感情が混在。句点の使い方が詩的に。
「四十七杯。三杯。数えられなくなった笑顔。」
→ 一文の長さ：3～5字の短句が並列。意図的な余白。

サキ文：「なんで」が疑問から共感へ変わる。
「なんで、そんなこと言うん」→「なんで、砂糖3杯なの」
→ 一文の長さ：8～15字。質問形が増える。

会話構造：沈黙（……）の行が増える。

【chapter12-13】危機・核心開示
キリエ文：「……」の沈黙が長くなる。「残滓だ」など重い定義が出現。
従来の「わからない」から「……そうだった」「……今は、違う」へ
→ 一文の長さ：5～12字。沈黙を挟む。

サキ文：初めて感情を直接言語化。ただし短く。
「死のうとするのをやめて。ずっと言いたかった。でも言えなかった。今言う」
→ 短い文の反復。リズム感。
→ 一文の長さ：5～20字。短文の蓄積による重み。

会話構造：「沈黙」が最も表情豊か。台詞より沈黙が主役。

【chapter14】静かな温かさ
キリエ文：「許容できる」が最後に返ってくる。意味が反転している。
短く、確定的。未来へ開かれた形。
「……許容できる」
→ 一文の長さ：5字。文のはじめの「……」が象徴的。

サキ文：「知ってる」が最後の言葉。これ以上何も言う必要がない。
「知ってる」
→ 一文の長さ：3字。完全な確定。

会話構造：台詞が最小化。ナレーションと沈黙が物語を運ぶ。
```

### 感情温度が上がるにつれての変化（具体例）

| 温度段階 | キリエの文 | サキの文 | 会話の特徴 |
|---|---|---|---|
| 冷（ch01） | 「致死的でなかった」 | 「いいけど」 | 平行線。相互作用なし |
| 温（ch06） | 「死ぶ選択肢がある。お前は生きている」 | 「覚えてて」 | 求める方向が見える |
| 熱（ch10） | 「この算術の答えは、まだわからない」 | 「知ってる」 | 同じ「わからなさ」の共有 |
| 沸（ch12） | 「お前が最初だ。五十万七千三十六回の間、一人もいなかった」 | 「もうやめて。死んでほしくない」 | 最初の直接対話 |
| 静寂（ch14） | 「……許容できる」 | 「知ってる。……どういたしまして」 | 言葉がほぼ不要。わかり合っている |

## ③ 新規サンプル台詞（各3本）
## 物語後半（chapter10-14）向け・既存台詞と重複なし

### キリエの新規台詞 × 3

**台詞①：「初めての比較」**
```
場面：chapter11、白蝕との対峙前夜。サキが眠った後、キリエが独白
----
「お前が来る前と、来た後。数字で比較するなら。
……死ぬまでの時間は、延びた。それだけだ。
違う。『延びた』ではない。『数え始めた』。」

用例・機能：
- 「延びた」という過去形から「数え始めた」の継続形へ転換
- 時間への向き合い方が「カウントダウン」から「カウントアップ」へ
- 感情を直接言わず、数と時間の「運用方法の変化」で示す
- 独白なので、聞き手がいない場面で使用可能
```

**台詞②：「拒絶できない理由」**
```
場面：chapter13、サキが「一緒にいて」と言う場面での返答
----
「理由がない。……破棄する理由が、ない。
お前が『いて』と言ったから。それだけの因果で、
足りてしまった。これは……初めてだ。
理由が『お前』だけで、充足する。」

用例・機能：
- 「破棄できない」から「破棄する理由がない」への深化
- 論理的な形をしながら、実は愛を言っている
- 「『お前』だけで充足する」＝「お前が全て」を意味する
- キリエが感情語を避けたまま、最も深い依存を示す表現
```

**台詞③：「数えない選択」**
```
場面：chapter14朝、砂糖3杯のコーヒーを飲みながら
----
「四十八杯目。……知っている。正確だ。
だが。今日は、数えない。
……数える必要がない。その数字が、何を意味するのか。
もう、わかったから。」

用例・機能：
- 「正確に数える」キリエが初めて「数えない」ことを選ぶ
- 理由が「わかったから」＝感情を言語化せずに理解した状態
- アーク完成の象徴。知識から叡智へ
- 最小限の台詞で、最も大きな変化を示す
```

### サキの新規台詞 × 3

**台詞①：「拒絶から許可へ」**
```
場面：chapter11、キリエが「死の話」をしようとしたとき
----
「……ちょっと待ってよ。
昨日だったら、『そんなこと話すな』って言ってた。
でも今は……聞きたい。全部。
なんで、そんなことになってるの。」

用例・機能：
- サキ自身が「自分の変化」を認識する瞬間
- 「拒絶」から「聞く」への転換を自分の言葉で表現
- 「なんで」が責める疑問形ではなく、相手を理解しようとする質問に
- サキが初めて「能動的に相手を知ろうとする」台詞
```

**台詞②：「消極的な愛情表現」**
```
場面：chapter12、キリエが「死にたい気持ちは消えた」と言った直後
----
「……うん。よかった。……本当にそう？
もう、試さないの？　もう、死ぬための方法、探さないの。」

用例・機能：
- 感情を直接言わず、相手の行動で愛を確認しようとする
- 「よかった」が唯一の感情語。通常は避けるが、この場で許可される
- 「試さないの」「探さないの」＝「ここにいつづけるの」を意味する
- サキらしい「消極的な愛の確認」
```

**台詞③：「習慣の深さ」**
```
場面：chapter14、朝のルーチンが定着した後の日常会話
----
「砂糖3杯。……毎日、そこに置いてある。
もう忘れたことない。手が覚えてる。
……何か、それが全部な気がして。」

用例・機能：
- 砂糖3杯という「小さな習慣」の深さを示す
- 「手が覚えてる」＝無意識化した愛情
- 「全部な気がして」＝この小さなことが全てを示している
- 知ってると言ってる。見てる。大切にしてる。全てが圧縮
```

## ④ 絶対言わないセリフ（追加分、各2本）

### キリエの追加禁止セリフ

**禁止①：「君は、失敗した」**
```
理由：
- キリエの論理では「失敗」は存在しない。結果があるだけ。
- サキの自殺未遂を非難する形になり、キリエのキャラに反する。
- キリエはサキを「評価」しない。観察するだけ。

代替表現：
「お前が、死を選ばなかった。その理由は、わからない。
だが、選ばなかった。それは、事実だ。」

【解説】：失敗ではなく「選択の結果」として述べる。評価を入れない。
```

**禁止②：「一緒にいてくれてありがとう」**
```
理由：
- 「ありがとう」は相互性の理解が必要。キリエはそれを学んでない（初期）。
- 「くれてありがとう」は感謝の感情を含む。キリエは感情語を避ける。
- 相手に感謝するということは、相手の「選択」を認識すること。
  キリエは「お前がそうしたから」という因果で説明するため、感謝の概念が出現しない。

代替表現：
「お前が、隣にいる。これが、変数だ。
お前が、隣にいなかったら。別の状態だった。
……その状態は、致死的だった。今はそうではない。」

【解説】：感謝ではなく、相手の存在による「状態変化」を述べる。
```

---

### サキの追加禁止セリフ

**禁止③：「キリエ、私、好きだよ」（直接的感情表白）**
```
理由：
- サキは感情を「言語化する習慣がない」
- 「好き」と言ったとたん、失うことへの恐怖が明確化される
- サキは「言わないことで距離を保つ」防衛機制を持っている

代替表現：
「……あ、キリエ。今朝、いなかった。
心臓がおかしくなった。悪い感じで。
だから、ちょっと……いてほしいな。」

【解説】：感情を言わず「相手の不在による身体反応」で示す。
サキは自分の感情に名前をつけない。代わりに「身体の変化」を報告する。
```

**禁止④：「何があっても、絶対に離さない」（未来の約束宣言）**
```
理由：
- サキは「今この瞬間」を生きるキャラ。未来を保証する言い方が合わない。
- 「絶対に」という確定は、自分の気持ちを過信すること。
  サキは自分の感情の強さを信頼していない（閉ざしてきたから）。
- 「離さない」という能動的な宣言は、サキの消極的性格に反する。

代替表現：
「……明日も、一緒にいてもいい？
その次の日も。その次も。」

【解説】：未来ではなく「今から明日への継続可能性」を問う形。
「いてもいい？」の問い形は、相手に判断を譲るサキらしさ。
毎日「続く」ことを願うが、約束はしない。
```

## ⑤ 二人の会話で「沈黙」が果たす役割

### 「……」の使い分け：キリエ × サキ

#### キリエの沈黙（「……」）が示すもの

| 沈黙の種類 | 意味 | 用例 |
|---|---|---|
| **開始の沈黙** | 言語を検索中。「わかる」の前段階 | 「……わかった」「……そうか」 |
| **考察の沈黙** | 論理を組み立てている。数を数えている | 「……四十七杯。三杯」（chapter10） |
| **感情未処理の沈黙** | 感情語を避けたまま、何か深いものを感じている | 「……お前が最初だ」（chapter12） |
| **確定の沈黙** | 決定した。これ以上の言葉は不要 | 「……許容できる」（chapter14） |
| **拒絶しない沈黙** | 「わからない」ことを受け入れた。相手を信じている | 「……」の後、相手の話を聞く |

**キリエの沈黙の特徴：**
- 短い。だいたい1～2秒。
- その後、必ず結論か確定が来る。
- 沈黙は「思考中」の表現。思考が完了すると言葉になる。

---

#### サキの沈黙（「……」）が示すもの

| 沈黙の種類 | 意味 | 用例 |
|---|---|---|
| **拒絶の沈黙** | 言いたくない。聞きたくない。壁を作っている | 「？」という顔で沈黙 |
| **言葉にならない沈黙** | 感情がありすぎて、言語化できない | chapter00の「泣く」シーン |
| **呼応の沈黙** | キリエの沈黙に応じている。一緒に考えている | 「……」「……」と二人で沈黙 |
| **承認の沈黙** | わかった。受け入れた。それ以上言う必要がない | 「ん。……」と頷く沈黙 |
| **懇願の沈黙** | 言葉では言えないが、隣にいてほしい。頼む | chapter04の「触れる」シーン |

**サキの沈黙の特徴：**
- 長い。3～5秒以上続くことも。
- その後、言葉が来るとは限らない。そのまま終わることもある。
- 沈黙は「感情中」の表現。感情が深いほど、言葉にならない。

---

### 沈黙が感情を表す具体的なシーンパターン × 3

#### **シーンパターン① 「理解の沈黙」**

**場面設定**
```
chapter11 夜。キリエが自分の正体を話した後。
サキが理解できない沈黙がある。
```

**シーン展開**
```
キリエ「……感情そのものが、形を持った。それが私だ」

サキ「……」（長い沈黙。目がキリエを見ている）

キリエ「わかるか」

サキ「……わかんない。でも……」（さらに沈黙）

キリエ「『でも』」

サキ「……でも、キリエなんだ。それだけ」
```

**沈黙の役割**
- サキの沈黙は「理解と非理解の狭間」を示す
- 科学的・理論的には理解できない（50万回の死、感情が形を持つ）
- しかし「感覚的には理解できている」という矛盾が沈黙に込められている
- キリエの沈黙は「相手が理解するのを待っている」という忍耐
- 二人の沈黙が「言葉を超えた通じ合い」の証

**執筆時のポイント**
- サキの沈黙の後に「でも、キリエなんだ」と来ることで、理屈を超えた受け入れが表現される
- キリエは「わかるか」と確認するが、サキの「わかんない」を受け入れる
- ここが「論理と感覚の相互補完」の場面

---

#### **シーンパターン② 「拒絶から傾聴への転換の沈黙」**

**場面設定**
```
chapter12 屋上から戻った夜。
キリエが「死のうとする理由」を説明しようとする。
```

**シーン展開**
```
キリエ「死のうとすることについて、話す」

サキ「……」（短い拒絶の沈黙。目を逸らしている）

キリエ「聞きたくないか」

サキ「……」（さらに長い沈黙。やがて目をキリエに向ける）

サキ「……聞く。全部。なんで、そんなことになってるの」
```

**沈黙の役割**
- 最初のサキの沈黙：「これ以上は聞きたくない」という拒絶
- 二番目のサキの沈黙：心の動き。「拒絶したい」と「知りたい」の葛藤
- 沈黙の中で「相手を理解したい」という気持ちが勝つ
- キリエは沈黙を「拒絶」と受け取るのではなく「思考中」と判断
- 二人の間に「相手を信じる時間」が生まれている

**執筆時のポイント**
- サキの沈黙の長さが「心の葛藤」を表現する
- 「拒絶の沈黙」と「受け入れの沈黙」は同じ「……」でも、前後の文脈で完全に異なる意味を持つ
- キリエが「聞きたくないか」と確認することで、相手の自由意志を尊重する姿勢が見える
- サキが「聞く」と言えるのは、キリエがそれを「強制しない」からこその信頼

---

#### **シーンパターン③ 「愛情の沈黙」（非言語的な確認）**

**場面設定**
```
chapter13 朝の一シーン。
キリエがコーヒーを飲んでいる。
サキがそばに座っている。
```

**シーン展開**
```
サキ「朝か」

キリエ「……そうだ」

サキ「……」（キリエを見ている。話しかけない）

キリエ「？」（サキが何を欲しているのか、わかってない）

サキ「……いや、何でもない。そこにいるんだ、って思って」

キリエ「……そうか」（わかった）

（二人とも、何も言わない。朝が続く）
```

**沈黙の役割**
- サキの最初の沈黙：相手を確認している。「そこにいる」ことの確認
- キリエの「……そうだ」：朝が来たこと。生きているという確定
- サキの「……」：言葉にならない喜び。感謝。安堵
- キリエの「わかった」という沈黙：相手を「理解した」という確定
- 最後の沈黙：二人で共有する時間。言葉が不要

**執筆時のポイント**
- この沈黙は「対話」ではなく「共在」を示している
- サキが「見ているだけ」でキリエが存在することを確認
- キリエが「わかった」と言うのは「サキの感情を論理化した」のではなく「サキの存在を受け入れた」
- 沈黙こそが、最も深い愛情表現である
- 物語が「言葉による愛」から「存在による愛」へシフトしていることを示す

## ⑥ 執筆時の実用ガイド

### キリエとサキの会話を書く時の3ステップチェック

**ステップ1：キリエの台詞を書いたら**
```
□ 感情語（好き、嫌い、嬉しい、つらい、悲しい）を使ってないか確認
□ 「……」で始まるか、或いは最小限形か
□ 理由を「～から」で説明しているか
□ 「許容」「記憶」「定義」など論理語彙を使ってるか
□ 相手（サキ）への言及があるか（キリエは相手がいないと話さない）
```

**ステップ2：サキの台詞を書いたら**
```
□ 感情を「直接」言ってないか（「好き」「支えます」は絶対NG）
□ 「けど」「なんで」「そう」で文が閉じてるか
□ 「行動を望む」形になってるか（「来て」「いて」「覚えてて」）
□ 「……」で沈黙を挟んでるか
□ 過去を「確定」させてないか（「～だった」より「～な気がして」）
```

**ステップ3：二人の対話を書いたら**
```
□ キリエの「わからない」とサキの「なんで」が噛み合ってるか
□ 沈黙（……）が「思考」か「感情」か区別されてるか
□ 一つの文が長すぎないか（特にサキ）
□ キリエが相手を「評価」していないか
□ サキが「覚悟」を決めてないか（進行形で良い）
```

### 後半（chapter10-14）で使える「ズレ」のコミカル場面

#### パターン①：「数」のズレ
```
キリエ「コーヒーを四十七杯飲んだ。砂糖は三杯。」
サキ「え、毎日砂糖3杯？　甘すぎでしょ」
キリエ「許容できる量をお前が決めた。四十七杯全て、三杯だ」
サキ「……決めたの俺？」
キリエ「そう」
サキ「改めても……いい？」
キリエ「……許容できない」（戸惑い）
```
→ キリエが「習慣=確定」と考えているため、変更を許容できない

#### パターン②：「社交辞令」の理解の遅さ
```
サキ「明日、買い物行く？」
キリエ「……理由があるか」
サキ「別に。一緒に行けたらな、って」
キリエ「『別に』とは、その事柄に重要性がない、という意味か。
　　　　　ならば、行く理由がない」
サキ「あ、違う。『別に行きたい』の『別に』じゃなくて……」
キリエ「人間の言語は、非効率だ」
```
→ 論理では「別に」は矛盾しているが、感覚では理解できない

#### パターン③：「身体」と「論理」のズレ
```
サキ「キリエ、今日も起きてくると思ってた」
キリエ「当然だ。昨日も起きた。その前も。パターンがある」
サキ「でも死ぬために出かけるんじゃなくて」
キリエ「……」（沈黙）
キリエ「その可能性は、排除した」
サキ「ほんと？」
キリエ「……根拠がない。だが。お前がそれを望まないから」
```
→ キリエは「論理で死を止める」ことができず「相手の望みで止める」しかない

### 最後のチェック：物語の核を失わないために

**キリエのキャラが壊れる危険な表現**
```
❌ 「君のことが好きだから」
❌ 「守ってあげたい」
❌ 「ずっと一緒にいたい」
✓ 「お前が、存在する。その理由は、わかっていない」
✓ 「破棄できない。お前が存在する限り」
✓ 「許容できる」（意味が反転している）
```

**サキのキャラが壊れる危険な表現**
```
❌ 「私、あなたを愛しています」
❌ 「一生、側にいます」
❌ 「あなたが支えです」
✓ 「いなくなるのは嫌だな」
✓ 「覚えてて」
✓ 「知ってる」（全てが圧縮されている）
```

**参考資料：二人のアークの最終形**

```
キリエ：「感情を知らずに変わった存在」
  → 最後まで「好き」という言葉は使わない
  → しかし「許容できる」の意味が「ここにいていい」に変わっている
  → 感情語を避けたまま、最も深い愛を示している

サキ：「感情を恐れたまま向き合った存在」  
  → 「好き」を言わないまま「覚えてて」で伝える
  → 「知ってる」一言で「見ていた・大切にしていた・愛していた」を示す
  → 消極的な言い方が、最も強い感情表現
```

**物語の最終メッセージ**
二人は「感情を言葉にする」ことで愛を確認するのではなく
「感情を言語化しないまま、存在で愛し合う」
その矛盾と違和感こそが、この物語の核心である。
